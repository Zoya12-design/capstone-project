# capstone-project

Capstone repository for the **AI Fluency** track (**FE-01: Environment and AI toolchain**). This repo holds environment setup, toolchain practice, and project work using Node.js, Git, and Cursor.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (LTS)
- [Git](https://git-scm.com/)

### Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/Zoya12-design/capstone-project.git
   cd capstone-project
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run the project**

   ```bash
   npm start
   ```
